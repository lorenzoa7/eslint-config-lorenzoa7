# Lorenzoa7 ESLint Config

## What is included in this package?

- Standard config
- React plugin
- React Hooks plugin
- Prettier for ESLint

## Setup

Install dependencies:
```
npm i -D eslint eslint-config-lorenzoa7
```
Inside `.eslintrc.json`:
```
{
  "extends": [
    "eslint-config-lorenzoa7", 
    "next/core-web-vitals"
  ]
}
```
